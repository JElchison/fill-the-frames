# fill-the-frames

This Python script will generate a worksheet containing 5 randomized "Fill the Frames" problems, in the style of https://bridges.mathlearningcenter.org/sites/default/files/documents/resources/HomeLearning/Grade4Bridges-HomeLearning-Unit2Mod1.pdf

<img width="1046" height="1505" alt="image" src="https://github.com/user-attachments/assets/92dca220-9c01-4e34-9f41-cacd15e0e08a" />

Use the Linux Bash script to send to a printer (fill in your printer name)
